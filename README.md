# WZKY
Small reverse proxy to help old Windows Servers with limited crypto support communicate securely.

## How To
Start the reverse proxy.
```shell
wzky --host <host>
```

Specify host port and bind port.
```shell
wzky --host <host> --host-port <port> --port <port>
```
